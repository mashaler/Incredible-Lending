Google Sheets integration

Paste your Google Apps Script Web App URL below. This allows applications submitted from any device to appear automatically in your dashboard.

📝 How to set up Google Sheets (one time)
Step 1. Go to sheets.google.com and create a new spreadsheet. Name it LendTrack Applications.
Step 2. Inside the spreadsheet, click Extensions → Apps Script.
Step 3. Delete all existing code and paste the script provided in your README or below.
Step 4. Click Deploy → New deployment. Choose type: Web app.
Step 5. Set Execute as: Me and Who has access: Anyone. Click Deploy.
Step 6. Copy the Web App URL and paste it above, then click Save.
Step 7. Paste the same URL into GOOGLE_SCRIPT_URL in lendtrack_apply.html and redeploy to Vercel.
